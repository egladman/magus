Graph Explorer

The magus knowledge graph, force-directed and searchable. New to magus? Read the docs or take the tour.

This view runs entirely in your browser. Run magus graph explore to open it seeded with your own workspace — your graph rides in the link's #fragment, which browsers never send to this server, so nothing leaves your machine. Install magus to try it.
