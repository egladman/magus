July 1, 2026 · Eli Gladman · Permalink

This is the first post on the magus docs blog. Posts land here alongside the docs, share the same rendering pipeline, and appear in the site search index.

New to magus? Start with the documentation, or try it live in the playground.

Try a runnable snippet

Click Run to execute this Buzz snippet in your browser. The WebAssembly interpreter is fetched lazily on the first click, so unopened pages stay light:

import "std";
final greeting = "hello from a runnable snippet";
std.print(greeting);
std.print("length is {greeting.len()}");

"Open in Playground →" carries the same snippet into /playground.html via a #source=<base64url> deep-link.
