About this interpreter

This is gopherbuzz, the Buzz interpreter magus uses to read magusfiles, compiled to WebAssembly. It runs entirely in your browser: no server, no network, nothing leaves this tab.

Installed on your machine, magus runs these targets for real. See the documentation or how to install.

What runs, what doesn't

  • Build tools don't run. A browser has no shell or filesystem, so run shows a target's plan (like magus run --dry-run) instead of invoking go-build, git, and friends.
  • Buzz is real. The magusfile is parsed and evaluated, so buzz ldflags(VERSION) returns -s -w -X main.version=1.4.0.
  • It stays local. Your edits never leave the tab; reload to start over.
Compiled with TinyGo to WebAssembly; the interpreter and your magusfile run entirely client-side.
