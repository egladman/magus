# BZZ1002: undefined type

A type annotation, cast, or literal referred to a type name that has no
definition in scope.

```text
[BZZ1002] buzz: line 3:12: undefined type "Poin"
  see: .../BZZ1002.md
```

## Resolution

- Check the spelling against the `object`/`enum` declaration.
- Declare the type if it does not exist, or import the module that declares it.
- Built-in types are `int`, `double`, `str`, `bool`, and the list/map forms;
  anything else must be declared.
