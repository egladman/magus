# BZZ1000: type error

A general type-check error. The buzz type checker found a problem the message
describes precisely, but the specific kind does not yet carry its own BZZ code.

```text
[BZZ1000] buzz: line 4:3: <the specific problem>
  see: .../BZZ1000.md
```

## Resolution

Read the message: it names the exact type problem (an unexpected type, a bad
operand, a member that does not exist). Fix the offending expression at the given
line and column. Common specific cases have their own codes - BZZ1001 (undefined
name), BZZ1002 (undefined type), BZZ1003 (non-bool condition), BZZ1004 (argument
error), BZZ1005 (type mismatch).
