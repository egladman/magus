# BZZ2001: unresolved import

An `import` did not resolve: buzz could neither find a built-in module of that
name nor a file at that path.

```text
[BZZ2001] buzz: import "helpers/math": module not found
  see: .../BZZ2001.md
```

## Resolution

- For a file import, check the path is correct relative to the importing file and
  that the file exists.
- For a module import, check the name against the modules buzz provides.
- The trailing detail (when present) names the underlying reason - a read error,
  a bad path - fix that specifically.
