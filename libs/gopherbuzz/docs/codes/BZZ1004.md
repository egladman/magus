# BZZ1004: argument error

A function call did not match the function's parameters. This covers the whole
family of call-shape mistakes:

- wrong argument count,
- a missing required argument,
- an unknown argument name,
- the same argument given twice,
- a positional argument after a named one.

```text
[BZZ1004] buzz: line 7:1: wrong argument count: got 1, want 2
  see: .../BZZ1004.md
```

## Resolution

Check the function's signature and match it: pass every required argument,
spell named arguments as the parameter names, and put all positional arguments
before any named argument.
